# coursetest Maven Project

该项目主要写课后练习

## coursetest project 目录结构说明

```
coursetes - src - main(test和main包结构对应) 
│  
├─string                    字符串相关练习
│    │ 
│    ├─TestHuiWenNum       字符串的反转(多种实现方式):StringBuilder:reverse()
│    ├─TestXiXueGuiNum     判断两个数组完全相同：Array.sort() Arrays.equals()
│                           
│      
├─workflow                  控制流程相关练习
│   │ 
│   ├─TestHuiWenNum        循环练习：*获取指定范围随机数方法总结
│      
├─utils                     工具类
│    │ 
│    ├─CheckUtils          数据校验工具类：包括字符串、数组的判空，字符串、数字大小比较等
│    ├─MyArrayUtils        数组工具类：数组排序：冒泡排序（两种方式）、快排实现
│    ├─MyStringUtils       字符串工具类：字符串反转（两种实现）      
│
